ここで適当に値をいじってから、テキストエディタなどで peta2_color.css を開き、生成された CSS を貼り付けて保存して下さい。 (これ単体では保存機能を持ちません。)

対象になるのは @media all の中身だけです。

もっと細かく変更したい場合は、CSS ファイルを自力で編集して下さい。

カラーピッカー等はご自身が使い慣れたものをご利用下さい。
